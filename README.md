# 🚀 Chinmay Deshpande | Portfolio

Welcome to my personal portfolio repository! This project showcases my journey as a Computer Science student at MIT Manipal, featuring my best projects across full-stack development, AI/ML, and embedded systems.

![Portfolio Preview](Chinmay.jpeg) *(Note: Placeholder for actual site screenshot)*

## 🛠️ Built With

- **Structure:** HTML5 (Semantic)
- **Styling:** Vanilla CSS3 (Custom design system, Responsive)
- **Logic:** Modern JavaScript (ES6+)
- **Animations:** GSAP / CSS Keyframes / Particle.js
- **Typography:** Outfit & JetBrains Mono (via Google Fonts)

## ✨ Key Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Dynamic Interactions:** Custom cursor, typed text animations, and hover-active cards.
- **Glassmorphism UI:** Modern semi-transparent elements with backdrop filters.
- **Dark Mode Aesthetic:** Premium dark theme with vibrant accents.
- **Project Showcase:** Interactive grid displaying featured and side projects.
